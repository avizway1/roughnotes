Create a repo and place the manifest file

1. Added the CodePipeline service role to EKS access entries:
  
   aws eks create-access-entry \
     --cluster-name ekswithavinash \
     --principal-arn arn:aws:iam::982424467695:role/service-role/AWSCodePipelineServiceRole-ap-south-1-eks-cicd-automation \
     --type STANDARD
   


2. Associated cluster admin policy:
  

   aws eks associate-access-policy \
     --cluster-name ekswithavinash \
     --principal-arn arn:aws:iam::982424467695:role/service-role/AWSCodePipelineServiceRole-ap-south-1-eks-cicd-automation \
     --policy-arn arn:aws:eks::aws:cluster-access-policy/AmazonEKSClusterAdminPolicy \
     --access-scope type=cluster
